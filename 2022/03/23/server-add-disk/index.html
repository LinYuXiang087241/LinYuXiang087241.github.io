通过H710 raid卡为服务器直通硬盘

  • 准备：
    • 服务器关机
    • 为服务器添加硬盘
  • 详细步骤：
    1. 重启服务器通过ctrl+R进入raid卡配置界面
    2. 进入raid卡H710管理界面，其中Unconfigured Physical Disk显示的便是新加入的硬盘
    3. 按F2进入操作，选择Create New VD
    4. 通过下图配置硬盘为raid0直通服务器
    5. 保存配置后，退出重启服务器。
    • 配置成功后会告知已经配置OK
    • 配置完成后的结果图
